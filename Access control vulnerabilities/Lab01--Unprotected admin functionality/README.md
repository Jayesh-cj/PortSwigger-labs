# Lab-1: [Unprotected admin functionality](https://portswigger.net/web-security/access-control/lab-unprotected-admin-functionality)

This lab has an unprotected admin panel. 

**Aim :-** Solve the lab by deleting the user ``` carlos ```. 



## Solution
- Go to ``` /robots.txt ```, this page will reveal path to the admin panel.

- Go to ``` /administrator-panel ```, the admin panel is now visible.

- Delete the user ``` carlos ```.
